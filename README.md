AngularJS 1 CRUD Example
===================

This is an AngularJS 1 CRUD Example without Database (using array of objects).
You can add, edit, delete, and search data.

##<a name="features"></a> Features

* Add data
* Edit data
* Delete data
* Search

##<a name="requirements"></a> You need to know

* HTML
* CSS
* Javascript
* Basic AngularJS

##<a name="howto"></a> How to run

* You can run directly from index.html or using server
